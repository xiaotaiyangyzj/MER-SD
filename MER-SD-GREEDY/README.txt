There are five data sets, and the data interpretation under each data set is the same.

dataset_answer_reword：The first column is the student number of the learner；The second column is the number of topics the learner has practiced；The third column shows whether the learners' answers to the exercises they have practiced are correct. The correct answer is 1 and the error is 0.

dataset_cognitive_status：Each row represents a learner, each column represents a knowledge point, and each element represents the learner's mastery of the exercise. The data is obtained by the neural CDM.
